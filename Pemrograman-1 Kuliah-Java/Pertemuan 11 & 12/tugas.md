# Tugas: Program Daftar Belanja (Menu Sederhana)
Deskripsi singkat  
Buat program berbasis menu untuk mengelola daftar belanja menggunakan Java. Pengguna dapat menambah, menghapus, dan menampilkan item sampai memilih keluar.

## Fitur
- Tambah Item
- Hapus Item
- Tampilkan Daftar Belanja
- Keluar

## Konsep Kunci
- ArrayList<String>
- Scanner
- while loop (untuk menampilkan menu berulang)
- switch-case (untuk menangani pilihan menu)

## Persyaratan
- Input harus divalidasi (mis. nomor item yang dihapus harus valid).
- Tampilan menu harus jelas dan user-friendly.
- Program berhenti ketika pengguna memilih opsi Keluar.

## Contoh Alur
1. Program menampilkan menu:
   1. Tambah Item
   2. Hapus Item
   3. Tampilkan Daftar Belanja
   4. Keluar
2. Pengguna memilih opsi (mis. 1) lalu memasukkan nama item.
3. Pengguna dapat menambah beberapa item, melihat daftar, atau menghapus item berdasarkan nomor.
4. Memilih 4 akan menghentikan program.

## Catatan Implementasi
- Gunakan ArrayList<String> untuk menyimpan item.
- Gunakan Scanner untuk membaca input dari pengguna.
- Gunakan loop (while/do-while) agar menu muncul berulang hingga pengguna memilih keluar.
- Gunakan switch-case untuk menangani setiap pilihan menu.
